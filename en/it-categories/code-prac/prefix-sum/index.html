前缀和

一维前缀和

主要思想：

  1. 求[l, r] l到r之间的和，比方说从第二个到五个数的总和可以表示为，前5个数的总和减去前一个数的总和：s[l~r] = s[r] - s[l - 1]，s为到第i个数的总和
  2. 默认s[0] = 0,来统一计算方式（节省了个if）
