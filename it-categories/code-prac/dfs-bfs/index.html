|阅读量:
特征 DFS BFS
数据结构 stack queue
储存空间 只用储存一条路的数值，所以跟高度或者路的长度有关 O(h)，使用空间较少 每次储存一层的数值，所以是指数增长 O(2^h)，使用空间较多
常用场景 不具有最短性 因为是一层一层地搜索，所以可以最快搜索到最短路径(特别是权重为1的时候)
